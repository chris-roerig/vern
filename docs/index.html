ver{n}

The version manager that just makes sense.

Install vern

$ curl -fsSL https://raw.githubusercontent.com/chris-roerig/vern/main/scripts/install.sh | bash click to copy

Paste that in a macOS Terminal or Linux shell prompt.

Go
Python
Node.js
Ruby
And more...

Smart versions

Install the latest, or be as specific as you want. vern install python 3 gets the latest 3.x.

Per-project versions

Drop a .vern file in your project. Version switching happens automatically.

No plugins needed

Go, Python, Node.js, and Ruby work out of the box. Add more with a simple YAML config.

Clean uninstall

Run vern implode and it's gone. Binary, config, installed versions ‚ÄĒ everything.

How it works

$ vern install go
Installing go 1.24.2...
Downloading go 1.24.2...
Successfully installed go 1.24.2
Set go 1.24.2 as default

$ vern install python 3.12
Installing python 3.12.8...
Building python 3.12.8 from source...
Successfully installed python 3.12.8

$ vern list
go (1 version):
  1.24.2 * (default)
python (1 version):
  3.12.8 * (default)

$ echo "python 3.12.8" > .vern  # pin version per project
